# Online Reservation and Booking System (Frontend)

This is the **frontend** of the Online Reservation and Booking System. The system allows administrators to manage multiple reservation types such as resorts, vehicles, conference halls, and more. Customers can easily make bookings, manage reservations, and perform online payments.  

This frontend is built with **React**, **TypeScript**, and **Tailwind CSS**.


## Features

- **Customer Features**
  - Browse and search for available reservation services
  - Make bookings easily with a simple interface
  - Manage existing bookings
  - Make secure online payments
  - View booking history

- **General Features**
  - Responsive design for mobile and desktop
  - Tailwind CSS for modern styling
  - Interactive UI components
